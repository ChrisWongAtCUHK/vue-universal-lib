# vue-universal-lib

## [How to create a Universal Library for Vue 2 & 3](https://dev.to/alvarosaburido/how-to-create-a-universal-library-for-vue-2-3-4lf8)

## Build and Install
1. build
```
yarn build
```
2. install
```
yarn add "~/vue-universal-lib"
```
3. Refer to main.ts and App.vue to use

## Project setup
```
yarn install
```
